# IvoBot 🤖
### Prompt Engineer Helper

O **IvoBot** é uma ferramenta para estruturar, testar e melhorar prompts antes de enviar para modelos de IA. Foco em clareza, contexto e repetibilidade.

---

## ✨ Experiência Sensorial
Pensado para reduzir tentativa e erro:

- **Editor de Prompts**: estruturação com validação.
- **Templates Prontos**: exemplos reutilizáveis.
- **Histórico**: compare versões e refinamentos.
- **Interface Direta**: use no browser sem setup extra.

---

## 🛠️ Funcionalidades Principais

| Recurso | Descrição |
| :--- | :--- |
| **Criação de Prompts** | Editor com campos objetivamente guiados. |
| **Templates** | Modelos base para tarefas comuns. |
| **Comparação** | Histórico e lista de versões. |
| **Entrega** | Saída pronta para uso em LLMs. |

---

## 🚀 Como Começar
1. Abra o arquivo principal no navegador.
2. Selecione um template ou crie um prompt novo.
3. Refine e teste antes de enviar ao modelo.

---
